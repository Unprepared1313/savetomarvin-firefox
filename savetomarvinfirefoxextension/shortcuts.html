This extension supports the following shortcuts:

  • Assign task to a category or project: type "#Work" (category), "#Midterm Paper" (project)

    • If your category/project includes multiple words, you must enclose them in quotations (e.g., #"Side Projects").

  • Assign task to a goal: type "##Get fit and healthy"

    • If your goal includes multiple words, you must enclose them in quotations (e.g., #"Side Projects").

  • Add a due date: type "due 2022-12-04", "due thursday", or "due 5d"

    • Non-specific dates such as "due next week" are not supported.
    • You must include both digits for dates, such as "due 2023-01-01".

  • Schedule task for a specific day: type "+tomorrow", "+2022-12-04"

  • Add a time by including it in the title anywhere: "8am Yoga class"

    • You must include both digits for dates, such as "due 2023-01-01".

  • Add a label: type "@easy"

  • Add time estimate: type "~30m" or "ca. 1h"

  • Add task to a specific section: type "!morning"

  • Add priority: type " *p1 " or " *p2 " or " *p3" or " *p0 "
